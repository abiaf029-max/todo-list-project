# To-Do List Project

A simple Python command-line To-Do List app for beginners.

## Features
- View tasks
- Add tasks
- Delete tasks
- Save tasks in a text file
- Load tasks when program starts

## Files
- main.py -> main Python program
- tasks.txt -> stores tasks
- README.md -> project information
- requirements.txt -> dependencies

## How to Run
1. Open terminal in project folder
2. Run:
   python main.py
