# 📝 To-Do List Project

A simple command-line **To-Do List application** built with **Python**.  
This project helps users manage daily tasks from the terminal by allowing them to add, edit, delete, and mark tasks as completed.

---

## 📌 Features

- View all tasks
- Add a new task
- Delete a task
- Mark a task as complete
- Edit an existing task
- Clear all tasks
- Save tasks in a text file
- Load saved tasks automatically when the program starts

---

## 🛠 Technologies Used

- **Python**
- File handling (`.txt` file)
- Functions
- Loops and conditionals
- Lists

▶️ How to Run the Project
Make sure Python is installed on your computer.
Open the project folder in VS Code.
Open the terminal.
Run the following command:
python main.py
📋 Menu Options

When you run the program, you will see options like:

Show tasks
Add task
Delete task
Mark task as complete
Edit task
Clear all tasks
Exit
💾 How It Works
Tasks are stored in tasks.txt
When the program starts, it loads previously saved tasks
Any new task changes are saved back into the file
